The plugin allows to give discounted product using coupon code.

== Description ==

The plugin allows to give discount product item and bogo discount using coupon code.

You can set discounts for products and product variations. You can also set discount based on categories.

Discounted product will be display on cart page when you applied coupon code.

== COUPON CODE TYPE AND FEATURES ==

* Free item based discount 
* Buy x and get y based discount 

== Examples ==

* Add any product or cetain product and get free product when coupon code applied.
* Buy 3 qty and get 1 qty as free when coupon code applied.

= Conditions based on =
Conditions can be based products, categories, items in cart.

*  All products
*  Specific products / Selected products
*  All categories
*  Selected categories


= Rules apply based on =

* Products
* Category

== Installation ==

Just use the WordPress installer or upload to the /wp-content/plugins folder. Then Activate the Woo Coupon Rules plugin.
More information could be found in the documentation

= Minimum Requirements =

* WordPress 5.6.1 or greater
* WooCommerce 3.0.0 or greater
* PHP version 7.4.0 or greater
* MySQL version 5.6 or greater
